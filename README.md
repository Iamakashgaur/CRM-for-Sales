# CRM Pro — Sales Task Manager

A single-file HTML CRM built for jewellery and B2B sales teams.

## Features

### Contacts
- 10,000+ contacts loaded from Zone Data (jewellers, panel types, zones)
- Search and filter by zone, city, category, follow-up status, assignee
- DNC (Do Not Call) marking

### Follow-Up Tracking
- Set next follow-up date and time
- Views for today's tasks, overdue, and hot leads
- Quick date shortcuts (+3 days, +1 week, +1 month)

### Call Logging
- Log calls with status and notes
- Auto call count badge per contact
- WhatsApp direct link per contact

### Team Management
- Admin creates users and assigns contacts
- Role-based access: admin sees all, agent sees own leads only

### Dashboard
- Stats: today / overdue / hot leads / DNC count
- CRM coverage progress
- Lead categories breakdown
- Recent activity feed

### Export
- Excel export (full data)
- PDF export

### Settings
- Import contacts via Excel
- User management
- Google Sheets sync
- Backup and restore

## Usage

Just open `index.html` in any browser — no server or installation needed.

## Tech
- Pure HTML, CSS, JavaScript (single file)
- No dependencies
